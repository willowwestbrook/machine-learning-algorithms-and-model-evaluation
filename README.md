# Machine Learning Algorithms and Model Evaluation

Comprehensive graduate-level machine learning course labs covering the full ML lifecycle — from exploratory data analysis through supervised and unsupervised learning, regularization, neural networks, and time series forecasting with LSTM models.

## Skills Demonstrated
- Regression diagnostics and assumption testing
- Regularization techniques (Ridge, Lasso, Elastic Net)
- Exploratory data analysis (EDA)
- Clustering and unsupervised learning
- Neural network design and training
- LSTM time series forecasting
- Model evaluation and validation methodology

## Tools & Technologies
- Python, scikit-learn, TensorFlow/Keras, pandas, NumPy, matplotlib, Jupyter Notebooks

## Contents
Organized lab subdirectories covering:
- **Regression Diagnostics** — residual analysis, assumption checking
- **Regularization** — Ridge/Lasso/Elastic Net implementations
- **EDA** — exploratory analysis workflows
- **Clustering** — k-means and hierarchical clustering
- **Unsupervised Learning** — dimensionality reduction and pattern detection
- **Model Training** — supervised learning pipelines
- **Neural Networks** — deep learning model construction
- **LSTM Time Series** — sequential forecasting models
- **Data Collection Assignment** — data sourcing and preparation
